<h3>🧪 Como utilizar a API com o Postman</h3> </br>

Para testar a API, siga os passos abaixo:

1 - Execute a aplicação Spring Boot -> DemoApplication.java (normalmente em http://localhost:8080).

2 - Utilize o Postman para realizar requisições HTTP.

Exemplo de endpoints:

GET http://localhost:8080/users
→ Retorna todos os usuários cadastrados.
